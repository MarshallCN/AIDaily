## 今日摘要

3 月 11 日的消息把行业的三条主线摆得很完整：一条是 AI cloud 和推理基础设施，一条是 agent 安全，一条是围绕更强 AI 系统的社会治理与公共讨论。三条线看起来分散，实际上都在围绕“AI 从实验走向真实系统”这个共同拐点。

## 关键动态

### 1. NVIDIA 与 Nebius 把 AI cloud 继续做大

- NVIDIA 与 Nebius 宣布深化合作，目标是扩展全栈 AI cloud，并把 AI factories、推理与 agentic AI 一起纳入长期建设框架。
- 这类合作不是单纯卖卡，更像是在提前锁定未来几年的 AI 基础设施容量和平台位置。

### 2. OpenAI 公开 agent 对抗 prompt injection 的设计思路

- OpenAI 把 prompt injection 描述得更接近社会工程问题，而不只是“识别恶意字符串”的过滤问题。
- 这意味着 agent 安全正在从 prompt engineering 走向系统安全：权限、确认机制、URL 安全、敏感信息外发控制都要一起做。

### 3. Anthropic 用 Institute 形式把长期影响研究单独拉出来

- Anthropic 宣布成立 The Anthropic Institute，目标是围绕更强 AI 带来的社会挑战提供更系统的研究输出。
- 这让“模型公司既做产品，也做治理叙事”的趋势更明显了。

### 4. Chrome 把 AI 浏览体验继续向更多地区放量

- Google 宣布把 Chrome 的 AI 体验扩展到印度、新西兰和加拿大，说明浏览器层的 AI 助手正在从试验功能进入更广泛分发阶段。
- 浏览器仍然是大量信息消费与工作入口，Chrome 的这类推进会继续抬高 AI 助手的日常渗透率。

## 为什么重要

- `AI Infra`、`AI 技术` 和 `行业动态` 不再是分开的栏目，它们现在会在同一天互相牵引。
- 未来几个月里，谁能同时解决推理成本、安全防线和分发入口，谁就更可能拿到下一阶段的产品优势。

## 明日跟踪

- Nebius 的合作会不会进一步带动更多 AI cloud 资本开支消息。
- OpenAI 会不会把更多 agent 安全机制做成对开发者可见、可配置的产品能力。
